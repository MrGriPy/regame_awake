# 🎮 ERA Game - React + Vite

Interface web interactive pour le jeu de plateau **ERA (REgame Awake)**.

🌍 **En ligne** : https://regame-awake.vercel.app

---
## Fonctionnalités
- Lancers de dés interactifs
- Boutique virtuelle avec dark patterns
-  Gestion des finances (cash vs carte)
-  Calcul automatique des scores
-  Dashboard animateur
-  Design responsive

## 🚀 Lancer en local

```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Ouvrir http://localhost:5173
